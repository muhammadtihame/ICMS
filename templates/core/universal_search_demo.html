3. Implementation Guide
HTML:
<input type="text" 
       data-universal-search="true"
       placeholder="Search...">
JavaScript (Optional - for custom initialization):
const searchInput = document.getElementById('my-search');
new UniversalSearch(searchInput, {
    minLength: 2,
    delay: 300,
    apiUrl: '/search-suggestions/'
});
